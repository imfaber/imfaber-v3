# Imfaber.me headless with Contenta CMS / JSON API and Vue.js

## GETTING STARTED

### REQUIREMENTS

- node v8.0.0 or >
- npm 5 or >

### BUILD SETUP

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
# service worker is disabled in dev
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# run unit tests. test resides in test/unit directory
$ npm test

# helper to deploy our app to github pages :
# you MUST have a local copy of "contentavuedemo.github.io"
$ npm run deploy
```
